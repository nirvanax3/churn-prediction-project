# Customer Churn Prediction System

## 📊 Overview

This project predicts customer churn using Machine Learning and visualizes results using Power BI.

## 🚀 Features

* Customer churn analysis dashboard (Power BI)
* Machine learning model (Scikit-learn)
* Prediction API (Flask)
* Backend integration (Spring Boot)
* Dynamic prediction visualization

## 🛠 Tech Stack

* Python (Scikit-learn, Pandas, Flask)
* Power BI
* Java (Spring Boot)
* Postman (API testing)

## ⚙️ How to Run

### 1. Run Flask API

```
python app.py
```

### 2. Send request (Postman)

POST http://localhost:8080/churn/predict

### 3. Refresh Power BI dashboard

## 📈 Output

* Churn Probability
* Risk Level
* Customer Prediction Insights
